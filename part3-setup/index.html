1Install Bun

If you don't have Bun installed yet, run this command:

curl -fsSL https://bun.sh/install | bash

Then reload your shell:

source ~/.bashrc # or ~/.zshrc

Verify installation:

bun --version

2Navigate to the React App

Open a new terminal window and navigate to the React app directory:

cd taproot-assets-webdev-examples/part3-react

3Install Dependencies

Install all required npm packages:

bun install
‚úì This will take 10-30 seconds
Bun will download React, TypeScript, Vite, and all dependencies

4Start the Development Server

Launch the React app:

bun dev
‚úì You should see:

‚ûú press h + enter to show help

üîß Troubleshooting

Gateway Connection Failed

curl http://localhost:8080/health

If this fails, restart the gateway:

cd ../taproot-assets-rest-gateway
cargo run

Port Already in Use

lsof -ti:5173 | xargs kill -9
bun dev

Bun Not Found

Make sure you reloaded your shell after installing:

source ~/.bashrc # or ~/.zshrc

No Assets Showing

Make sure you:

  • Completed Part 2 and minted some assets
  • Waited ~30 seconds for blockchain confirmation
  • Have auto-mine enabled in Polar (30s interval)
